## 主要变化

### 1. 新增：Outbound Hook 系统 (v2.1.9)

**新增文件**: `src/messaging/outbound-hooks.ts`

为所有外发消息路径添加了两个插件 hook：

| Hook | 触发时机 | 功能 |
|------|----------|------|
| `message_sending` | 发送前 | 拦截/修改/取消消息 |
| `message_sent` | 发送后 | 通知（fire-and-forget） |

**影响范围**:
- `sendWeixinOutbound` ([channel.ts](src/channel.ts))
- `process-message` 的 `deliver` 回调
- `sendMedia` 路径

**代码示例**:
```typescript
// 发送前拦截
const result = await applyWeixinMessageSendingHook({
  to: ctx.To,
  text: filteredText,
  accountId: deps.accountId,
  mediaUrl,
});
if (result.cancelled) {
  logger.info(`outbound: cancelled by message_sending hook`);
  return;
}

// 发送后通知
emitWeixinMessageSent({
  to: ctx.To,
  content: text,
  success: true,
  accountId: deps.accountId
});
```

### 2. 新增：Bot Agent 标识 (v2.4.2)

**API 类型变化**: [src/api/types.ts](src/api/types.ts)

```typescript
export interface BaseInfo {
  channel_version?: string;
  /**
   * 自我标识，类似 HTTP User-Agent
   * 来源: channels.openclaw-weixin.botAgent
   * 默认: "OpenClaw"
   * 用途: 可观测性（日志、监控聚合）
   */
  bot_agent?: string;
}
```

**新增 API 端点**:
- `POST ilink/bot/msg/notifyStop` — 通道停止通知
- `POST ilink/bot/msg/notifyStart` — 通道启动通知

**Bot Agent 格式规范** ([src/api/api.ts](src/api/api.ts)):
```
bot_agent = product *( SP product )
product   = name "/" version [ SP "(" comment ")" ]
name      = 1*32( ALPHA / DIGIT / "_" / "." / "-" )
version   = 1*32( ALPHA / DIGIT / "_" / "." / "+" / "-" )
comment   = 1*64( printable ASCII minus "(" ")" )
```

**限制**:
- 最大长度: 256 字节
- 编码: ASCII / UTF-8
- 失败时回退: `"OpenClaw"`

### 3. 重构：移除 runtime.ts (v2.4.2)

**删除文件**: `src/runtime.ts`

**v2.1.8 方式**:
```typescript
// 动态获取 channelRuntime
const pluginRuntime = await waitForWeixinRuntime();
channelRuntime = pluginRuntime.channel;
```

**v2.4.2 方式**:
```typescript
// Gateway 直接注入
startAccount: async (ctx) => {
  if (!ctx.channelRuntime) {
    throw new Error("ctx.channelRuntime missing — host too old");
  }
  // 直接使用 ctx.channelRuntime
}
```

**原因**: 简化依赖注入，让 gateway 控制生命周期。

### 4. API 层改进 (v2.4.2)

**[src/api/api.ts](src/api/api.ts)** 变更:

```typescript
async function apiPostFetch(params: {

  endpoint: string;
  body: string;
  token?: string;
  timeoutMs?: number;  // 可选
  label: string;
}): Promise<string>
```

### 5. 通道生命周期钩子 (v2.4.2)

**[src/channel.ts](src/channel.ts)** 新增:

```typescript
export const weixinPlugin: ChannelPlugin = {
  // ...

  stopAccount: async (ctx) => {
    const resp = await notifyStop({
      baseUrl: account.baseUrl,
      token: account.token,
    });
    // 失败时仅记录警告
  },

  startAccount: async (ctx) => {
    await notifyStart({
      baseUrl: account.baseUrl,
      token: account.token,
    });
    // ... 启动监控
  }
}
```

**用途**: 通知微信服务器通道状态变化。

## 兼容性影响

| 变更 | 影响程度 | 说明 |
|------|----------|------|
| Outbound hooks | ✅ 向后兼容 | 可选功能，无 hooks 时跳过 |
| Bot Agent | ✅ 向后兼容 | 默认值填充，服务器端可选 |
| 移除 runtime.ts | ⚠️ 破坏性 | 需要 OpenClaw >= 2026.3.22 |
| notifyStart/Stop | ✅ 向后兼容 | 失败时仅记录警告 |
| dist/ 目录 | ✅ 向后兼容 | 预编译输出，不影响源码 |

- OpenClaw >= 2026.3.22
